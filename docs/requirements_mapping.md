# Requirements Mapping - MindfulBot Project

| User Story / Requirement | Page Component | Implementation Detail |
| :--- | :--- | :--- |
| **View Breathing Exercises** | `Breathing.vue` | Guided circular breathing animation with focus timers. |
| **Get AI Logical Support** | `AiSupport.vue` | Chat interface featuring analytical mode and session insights. |
| **Analyze Mood Patterns** | `AiSupport.vue` | Triggered via `getAnalysis()` using `aiService` analytics. |
| **Read Wellness Research** | `Wellness.vue` | Searchable knowledge base with category filtering. |
| **Navigate Mental Space** | `Navbar.vue` | Global navigation across Home, AI, Wellness, and Breathing. |
| **Access Actionable Data** | `aiService.js` | Logic engine providing Eisenhower Matrix & 5-Minute Rule frameworks. |
