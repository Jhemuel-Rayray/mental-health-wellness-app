## Project Description
A modern wellness awareness platform designed to provide systematic, analytical support for mental health. Unlike traditional bots, this companion uses **Logical Intelligence** to help users deconstruct stress, identify cognitive distortions, and apply actionable productivity frameworks.

### Key Intelligence Features:
* **Analytical Session Insights**: Real-time tracking of emotional and logical patterns.
* **Strategic Frameworks**: AI-driven suggestions including the Eisenhower Matrix and the 5-Minute Rule.
* **Biosystem Audit**: Analysis of the physical-mental link (sleep, hydration, and focus).
